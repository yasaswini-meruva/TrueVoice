# TrueVoice — Know What's Real

An AI-powered content analysis platform that detects AI-generated text, identifies writing patterns, humanizes content, and compares authorship.

## 📋 Overview

TrueVoice leverages the Groq API with LLaMA-3.3-70B to analyze and verify the authenticity of content. The application provides a modern, intuitive interface for content verification.

## ✨ Features

- AI Detection
- Model Fingerprinting
- Text Humanization
- Voice Match (Authorship Verification)
- Analysis History
- Real-time LLaMA-powered analysis

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python (SimpleHTTPServer)
- **AI API**: Groq API (LLaMA-3.3-70B)

## 📦 Prerequisites

- Python 3.x
- Groq API key ([Get one here](https://console.groq.com/keys))
